ss2url -- Screen Shot To URL
============================

## ss2dropbox.sh

Shell script to upload screenshots to Dropbox and put share URL on pasteboard.

Requires [dropbox_uploader.sh](https://github.com/andreafabrizi/Dropbox-Uploader/blob/master/dropbox_uploader.sh)

## ss2url.sh

Shell script to upload screenshots to your own server via `scp`

## Growl

Both scripts use [Growl](http://growl.info) and [growlnotify](http://growl.info/downloads#generaldownloads).
